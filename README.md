# 🚀 CoinEx Pump Radar AI

AI-based Web App for detecting early pump signals in low-cap meme coins on CoinEx.

---

## 🧠 Features

- Real-time market scanning (CoinEx API)
- Pump DNA scoring system
- Top 3 coin signal generator
- Web API endpoint (`/signals`)
- Lightweight Flask backend

---

## ⚙️ Tech Stack

- Python 3
- Flask
- Requests
